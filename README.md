# Criptoaritmo

Este programa sera un proyecto donde juntaremos una intefaz hecha en ruby
mientras que el backend sere hecho por medio del lenguaje prolog. Ambos se tendran una conexión
mediante sockets.

## Instalación de librerias

```bash
gem install libui
```

## Ejecución

*Servidor*

```bash
ruby Front/front.rb
```

*Cliente*

```bash
swipl Servi/Servi.pl
```
